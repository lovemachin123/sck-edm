분류 방법

5개 영역 점수 중 상위 2개 축을 먼저 고르고, 평균 강도와 1-2위 점수 차이, 3위 영역의 보조 강도를 함께 반영해 세부 타입을 나눕니다.

그래프 해석

분포가 높은 타입은 현재 고객군에서 반복적으로 나타나는 공통 니즈입니다. 개별 상담보다 세미나, 템플릿, 패키지 제안으로 묶는 편이 효율적입니다.

제안 방향

상위 타입의 핵심 불편을 먼저 언급하고, AutoCAD Plus나 Web을 기능 소개가 아닌 업무 개선 시나리오로 연결하면 설득력이 높아집니다.

자동화/확장성 비중이 높을 때 LISP, 서드파티, ERP/Excel 연동을 기준으로 현재 반복 작업 시간을 수치화하고 AutoCAD Plus 중심의 표준화 제안을 준비합니다.
협업/이동성 비중이 높을 때 최신 도면 공유, 마크업, 현장 확인, 버전 혼선을 먼저 짚고 AutoCAD Web 기반의 검토 흐름을 제안하는 것이 유리합니다.
설계 전문성 비중이 높을 때 표준 부품, 공차, 은선, Hole Table처럼 실무자가 바로 체감하는 전문 기능 데모를 앞세우면 전환 가능성이 커집니다.
